서버

1. 클라이언트들이 한번에 여러명이 접속해서 서로 채팅으로 소통할 수 있도록 한다. o

2. 화면으로 현재 클라들의 채팅 현황을 볼 수 있게한다. o

3. 클라가 접속을 종료하면 그것을 다른 클라이언트들도 볼 수 있도록 알려준다. o

4. 서버가 닫히는 순간 해당 서버가 열렸던 동안의 채팅로그를 .txt파일로 남긴다.

5. 채팅방에 파일을 올리고 해당파일이 이미지 파일이라면 클릭했을 때 미리보기를 할 수 있도록한다.



클라

1. 닉네임 설정한 뒤 접속버튼 누르면 닉네임이 적용되는 시스템. o

2. 화면으로 현재 클라들의 채팅현황 볼 수 있음. o

3. 접속 종료하면 접속하는 동안의 채팅기록을 txt파일로 남긴다.

5. 채팅방에 파일을 올릴 수 있고, 해당파일이 이미지파일이면 내가 로컬에 파일이 없어도 클릭했을 때 해당 이미지를 미리보기 할 수 있도록한다.
